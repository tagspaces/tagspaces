        Search Options  Include files from the subfolders (by adding questionmark '?' in front of the query)
         Including "!" in front of a term (e.g. !term) will exclude all files with this term in their filenames.
         Including "+" in front of a term (e.g. +tag) will include all files tagged with this tag.
         Including "-" in front of a term (e.g. -tag) will exclude all files tagged with this tag.
         All terms should be true in order the file to appear in the search results.
         The searched term should be part of the filename. Currently the content of the file is ignored.
